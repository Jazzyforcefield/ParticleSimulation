UCS Shortest Path

Shows an animated agent traversing a path found by UCS. Currently no collision checking or path highlighting due to time constraints. The path is printed in the output.
Each milestone has 5 neighbors, which is a parameter that can be altered in the simulation within the configuration.cpp file.
